# Celeste 64: Fragments of the Mountain (for PortMaster)

**Unofficial Native Port for R36S, ArkOS, and Retro Handhelds.**

This is a native ARM64 build of *Celeste 64*, originally created by Maddy Makes Games for the 6th anniversary of Celeste. This port was recompiled using **NativeAOT** to run flawlessly on low-power devices like the R36S, avoiding the stuttering issues of Mono.

![Celeste 64](https://img.itch.zone/aW1nLzE0NzY2NTE2LmdpZg==/original/Xw5%2B10.gif)

### üéÆ Features & Optimizations
* **Native Performance:** Recompiled to native machine code (No Mono runtime overhead).
* **Resolution Fix:** Logic adjusted for **320x240 (4:3)** screens with perfect integer scaling.
* **Controls Mapped:**
    * **A / B:** Jump
    * **X / Y:** Dash
    * **Start:** Pause

---

### üì• How to Install

1.  Go to the **[Releases Page](../../releases)** and download the latest `celeste64.zip`.
2.  Unzip the content. You will see a folder named `celeste64` and a script `Celeste64.sh`.
3.  Copy both to your device's ports folder (e.g., `EASYROMS/ports` or `roms/ports`).
4.  Update your PortMaster libraries if the game does not launch.
5.  Launch via the **Ports** menu on your device.
